Регистрация нового пользователя (юридическое лицо) с вводом латинских букв в поле "Наименование банка" (данные о счете организации)

* Тестовые данные: 

Предусловия:

1. Создать юзер аккаунт с электронной почтой

2. Создан тестовый электронный ящик(и)

3. Открыта форма регистрации и выбрана вкладка "Юридическое лицо"
   (ссылка на прод [Регистрация](https://stroyrem-nn.ru/user/register) и на тест [Регистрация](https://test2.stroyrem-nn.ru/user/register))

4. Все поля заполнены валидными данными, кроме поля "Наименование банка" (данные о счете организации)
* Шаги:
1. В поле "Наименование банка" (данные о счете организации) ввести название банка состоящее из латинских букв

2. Нажать кнопку "Зарегистрироваться"
   
   **ОР:** Регистрация проходит успешно и открывается страница профиля пользователя
* Постусловие: выйти из профиля и удалить тестовые данные на тестовом сервере
